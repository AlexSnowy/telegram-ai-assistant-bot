# 🚀 Быстрый старт - AI Assistant Telegram Bot

## ✅ Всё готово к запуску!

### 🆕 **Новые функции: Регистрация и выбор языка**

1. **При первом запуске** бот предложит выбрать язык:
   - 🇷🇺 Русский
   - 🇺🇿 O'zbekcha
   - 🇺🇸 English

2. **Обязательная регистрация** через номер телефона:
   - Бот запросит поделиться контактом
   - Номер сохраняется в Firestore
   - После регистрации - доступ ко всем функциям

3. **Мультиязычный интерфейс** на всех этапах

### 1️⃣ **Запуск бота (локально)**

**Windows:**
```cmd
run.bat
```

**Или вручную:**
```cmd
.venv\Scripts\activate
python -m src.bot
```

**Linux/Mac:**
```bash
./run.sh
```

### 2️⃣ **Где размещать файлы:**

#### 📝 **Промты** (добавлять через Telegram или в папку `prompts/`)
- Формат: `.txt` файлы
- Пример: `prompts/example_china_trade.txt`
- Загружайте через Telegram: отправьте `.txt` файл боту

#### 📚 **Документы для базы знаний** (добавлять через Telegram или в папку `knowledge_base/`)
- Поддерживаемые форматы: `.txt`, `.docx`, `.xlsx`, `.pdf`
- Загружайте через Telegram: отправьте файл боту
- Или скопируйте в папку `knowledge_base/`

### 3️⃣ **Команды бота:**

| Команда | Описание |
|---------|---------|
| `/start` | Начало работы, приветствие |
| `/help` | Помощь и инструкции |
| `/addprompt` | Добавить промт (отправьте .txt файл) |
| `/listprompts` | Показать все промты |
| `/knowledge` | Поиск в базе знаний |
| `/addknowledge` | Добавить документы (отправьте файл) |
| `/status` | Статус системы |

### 4️⃣ **Тестирование:**

1. Откройте Telegram
2. Найдите вашего бота (по токену)
3. Отправьте `/start`
4. Бот ответит на узбекском языке
5. Загрузите документы через `/addknowledge`
6. Задавайте вопросы!

### 5️⃣ **Деплой на Google Cloud Functions:**

```bash
chmod +x deploy.sh
./deploy.sh
```

После деплоя:
```bash
curl -X POST "https://api.telegram.org/bot<ВАШ_ТОКЕН>/setWebhook?url=<URL_ФУНКЦИИ>"
```
