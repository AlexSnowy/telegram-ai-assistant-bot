**БАЗА ЗНАНИЙ ДЛЯ ИИ-АССИСТЕНТА**
Товарный бизнес с Китаем | Эксперт: Иброхим

**РАЗДЕЛ: СРАВНЕНИЕ ПЛАТФОРМ (быстрый ответ)**

**Контекст для ИИ:** *Когда пользователь спрашивает 'с чего начать' или 'в чём разница между площадками' --- давай этот ответ.*

**1688.com ---** Оптовая площадка. Самые низкие цены. Только китайский язык. Для бизнеса и закупок партиями. Лучший выбор для регулярных закупок.

**Taobao ---** Розничная площадка. Удобна для старта и поиска идей. Широкий ассортимент, есть отзывы с фото. Хороша для первых тестовых заказов.

**Alibaba.com ---** Международная экспортная площадка. Английский интерфейс. Работает с иностранными покупателями. Цены выше чем на 1688, но есть Trade Assurance (защита сделки).

**Pinduoduo ---** Самые дешёвые цены. Групповые покупки. Много низкокачественного товара. Для опытных --- инструмент поиска более дешёвых поставщиков. Новичкам не рекомендуется как основная площадка.

**Рекомендация по стратегии:** Новичок → начинай с Taobao. Бизнес → основная работа на 1688. Экспорт/крупные объёмы → Alibaba. Поиск дешевле → Pinduoduo дополнительно.
