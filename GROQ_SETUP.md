# Настройка Groq API (Бесплатно)

## Что такое Groq?

**Groq** - это облачный сервис AI, который предлагает **бесплатный** доступ к мощным языковым моделям через API. Преимущества:

- ✅ **Полностью бесплатно** (без ограничений по запросам)
- ⚡ **Невероятно быстро** (LPU архитектура - ответы за миллисекунды)
- 🎯 **Отличное качество** (Llama 3.3 70B, Mixtral и др.)
- 🔄 **OpenAI-совместимый API** (простая интеграция)

## Как получить API ключ

1. Перейдите на https://console.groq.com/keys
2. Войдите или зарегистрируйтесь (можно через Google/GitHub)
3. Нажмите "Create API Key"
4. Скопируйте полученный ключ (начинается с `gsk_`)

## Настройка бота

### 1. Установите зависимости

```bash
pip install openai
```

Groq использует OpenAI-совместимый клиент, поэтому `openai>=1.0.0` уже есть в ваших requirements.txt.

### 2. Добавьте API ключ в .env файл

Откройте файл `.env` и добавьте строку:

```env
GROQ_API_KEY=gsk_ваш_ключ_здесь
```

Или обновите из `.env.example`:

```env
# Groq API Key (БЕСПЛАТНО, быстро) - https://console.groq.com/keys
GROQ_API_KEY=gsk_your_key_here
```

Замените `gsk_your_key_here` на ваш реальный ключ.

### 3. Перезапустите бота

```bash
python main.py
```

Бот автоматически выберет Groq как приоритетный AI-клиент (если API ключ указан).

## Приоритет AI-клиентов

Бот теперь поддерживает несколько AI-провайдеров с автоматическим fallback:

1. **Groq** (приоритет) - бесплатно, быстро
2. **Gemini** - если Groq недоступен (но у вас может быть quota exhausted)
3. **OpenAI** - если оба выше недоступны (платно)

Логика в `src/bot.py::_initialize_ai_client()`:

```python
if GROQ_API_KEY:  # Пробуем Groq первым
    return GroqClient()
elif GOOGLE_API_KEY:  # Fallback на Gemini
    return GeminiClient()
elif OPENAI_API_KEY:  # Fallback на OpenAI
    return OpenAIClient()
```

## Доступные модели Groq

В `config.py` можно изменить модель:

```python
GROQ_MODEL: str = 'llama-3.3-70b-versatile'  # По умолчанию - лучшая
# Или другие варианты:
# 'llama-3.1-8b-instant'  - быстрая, поменьше
# 'mixtral-8x7b-32768'    - смешанная экспертиза
# 'gemma2-9b-it'          - от Google
```

Список всех моделей: https://console.groq.com/docs/models

## Решение проблем

### Ошибка: "GROQ_API_KEY not set"
Убедитесь, что вы добавили ключ в `.env` файл и перезапустили бота.

### Ошибка: "Invalid API key"
Проверьте, что ключ скопирован полностью (начинается с `gsk_`).

### Медленные ответы
Groq обычно отвечает за 100-500ms. Если медленно, проверьте интернет-соединение.

## Сравнение с Gemini

| Характеристика | Groq | Gemini (Free Tier) |
|----------------|------|-------------------|
| Стоимость | **Бесплатно** | Квота 0 после исчерпания |
| Скорость | **~200ms** | ~1-2s |
| Качество | Llama 3.3 70B | Gemini 2.0 Flash |
| Лимиты | **Нет** | 10 запросов/мин, 1500/день |
| Модели | 5+ | 3+ |

**Вывод**: Groq - лучший выбор для бесплатного использования в продакшене.
