# Сравнение бесплатных AI API

## Рекомендация: Groq (Лучший выбор)

### ✅ Преимущества:
- **100% бесплатно** без лимитов
- **Скорость**: ~200ms (самый быстрый)
- **Качество**: Llama 3.3 70B (SOTA)
- **Простота**: OpenAI-совместимый API

### 📝 Регистрация:
https://console.groq.com/keys

### 💡 Использование:
```python
from openai import OpenAI

client = OpenAI(
    api_key="gsk_your_key",
    base_url="https://api.groq.com/openai/v1"
)

response = client.chat.completions.create(
    model="llama-3.3-70b-versatile",
    messages=[{"role": "user", "content": "Привет!"}]
)
```

## Другие бесплатные варианты

### 1. Google Gemini (Free Tier)

**⚠️ Проблема**: У вас quota exhausted (0 лимитов)

- **Лимиты**: 10 запросов/мин, 1500/день (если доступно)
- **Скорость**: ~1-2s
- **Качество**: Хорошее
- **Модели**: gemini-1.5-flash, gemini-1.5-pro

**Регистрация**: https://makersuite.google.com/app/apikey

**Проблема в вашем случае**: У вас квота 0, значит либо:
- Вы исчерпали дневной лимит
- Проект не активирован для free tier
- Нужно создать новый Google Cloud проект

---

### 2. Hugging Face Inference API

**Лимиты**: 10k токенов/месяц бесплатно

- **Скорость**: ~1-3s
- **Качество**: Зависит от модели
- **Модели**: Любые с HF Hub

**Регистрация**: https://huggingface.co/settings/tokens

**Использование**:
```python
import requests

API_TOKEN = "hf_your_token"
response = requests.post(
    "https://api-inference.huggingface.co/models/meta-llama/Llama-2-70b-chat-hf",
    headers={"Authorization": f"Bearer {API_TOKEN}"},
    json={"inputs": "Привет!"}
)
```

**Минусы**: Медленный cold start, нужно ждать загрузки модели.

---

### 3. OpenRouter

**Лимиты**: $1-3 бесплатно в месяц (зависит от акции)

- **Модели**: Доступ к разным провайдерам через один API
- **Качество**: Разное

**Регистрация**: https://openrouter.ai/keys

**Использование**:
```python
from openai import OpenAI

client = OpenAI(
    api_key="sk-or-v1-your-key",
    base_url="https://openrouter.ai/api/v1"
)
```

**Минусы**: Мало бесплатных кредитов, нужна карта для верификации.

---

### 4. Perplexity AI

**Лимиты**: 5 запросов/4 часа (бесплатно)

- **Качество**: Хорошее, с поиском в интернете
- **Модели**: sonar, sonar-pro

**Регистрация**: https://www.perplexity.ai/settings/api

**Минусы**: Мало запросов, не подходит для бота.

---

### 5. Together AI

**Лимиты**: $25 кредитов на 90 дней

- **Модели**: Много open-source моделей
- **Качество**: Хорошее

**Регистрация**: https://together.ai/

**Минусы**: Нужна карта, кредиты заканчиваются.

---

### 6. Replicate

**Лимиты**: ~10k токенов/месяц

- **Модели**: Множество, включая Stable Diffusion
- **Качество**: Хорошее

**Регистрация**: https://replicate.com/account/api-tokens

**Минусы**: Медленный, лимиты маленькие.

## Рекомендация для вашего бота

### ✅ **Используйте Groq** - вот почему:

1. **Бесплатно без ограничений** - можно использовать в продакшене без страха исчерпать квоту
2. **Скорость** - в 5-10 раз быстрее Gemini
3. **Качество** - Llama 3.3 70B на уровне GPT-4 по многим задачам
4. **Простота** - API совместим с OpenAI, у вас уже есть OpenAI клиент
5. **Надежность** - нет квот, нет скрытых ограничений

### Как подключить:

1. Получите ключ: https://console.groq.com/keys
2. Добавьте в `.env`: `GROQ_API_KEY=gsk_...`
3. Перезапустите бота

**Готово!** Бот автоматически начнет использовать Groq.

## FAQ

### Q: Нужно ли менять код?
**A:** Нет, код уже готов. Просто добавьте `GROQ_API_KEY` в `.env`.

### Q: Сколько можно использовать?
**A:** Без ограничений. Groq не имеет лимитов на запросы.

### Q: Какая модель лучше?
**A:** `llama-3.3-70b-versatile` - лучшая по качеству. Для скорости можно `llama-3.1-8b-instant`.

### Q: Будет ли работать в продакшене?
**A:** Да, Groq production-ready и используется в реальных проектах.

### Q: Что если Groq упадет?
**A:** Бот автоматически переключится на Gemini или OpenAI (если ключи есть).
